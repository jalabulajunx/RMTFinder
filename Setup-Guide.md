# ⚙️ Setup Guide

### 📥 Download & Run

1. Open Android Studio
2. `File > Open` → Select project folder
3. Click ▶️ Run to build on emulator or device

### ✅ Requirements

- Android Studio (Electric Eel or later)
- JDK 11+
- Android SDK 21+

### 🔑 Permissions

App requests:
- Internet access
- Location access (for "Near Me")
